# @alexrebula/giselle-ui

A standalone React + TypeScript UI component library. Independent theming via CSS Custom Properties. No MUI. Publishable and usable without any external theme provider.

## What is in this documentation?

- **[Styling Roadmap](./styling-roadmap)** — CSS Custom Properties strategy, Phase A (CSS Modules), Phase B (Tailwind), decision table.

## Install

```bash
npm install @alexrebula/giselle-ui
```

## Peer dependencies

```bash
npm install react react-dom
```
